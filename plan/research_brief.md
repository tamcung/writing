# 论文总纲

## 基本信息

- 论文类型：中文硕士毕业论文
- 学科定位：网络空间安全方向
- 论文题目：面向语义保持变形的鲁棒 SQL 注入检测方法研究与实现

## 中心问题

现有 SQL 注入检测器在语义保持变形和外部分布偏移场景下存在鲁棒性不足的问题。传统同分布测试中的高指标不能充分反映检测器面对混淆、变形和未见样本时的实际检测能力。

## 核心主张

本文以语义保持变形场景下的 SQL 注入鲁棒检测为目标，提出一种基于配对表示学习的检测方法，并实现相应原型系统。

## 方法收敛

- 主方法：`pair_proj_ce`
  - 含义：基于配对样本的投影表示分类方法
- 扩展机制：`pair_canonical`
  - 作为表示对齐扩展和补充实验保留
  - 不作为全文 headline

## 研究问题

1. 现有 SQL 注入检测模型在语义保持变形下是否存在显著鲁棒性下降现象。
2. 仅依赖数据增强是否足以提升检测器鲁棒性，还是还需要显式的表示学习约束。
3. 如何在保证检测性能的同时，实现一个可运行的 SQL 注入检测原型系统。

## 预期贡献

1. 构建面向语义保持变形场景的 SQL 注入鲁棒性分析与评估框架。
2. 提出一种基于配对表示学习的鲁棒 SQL 注入检测方法，以提升模型在变形样本与外部分布样本上的泛化能力。
3. 设计并实现 SQL 注入检测原型系统，并通过多模型对比实验验证所提方法的有效性。
